# Harambee Stars Manager

A modern community fundraising platform designed for transparency and ease of use. This application allows community members to manage multiple campaigns and simulate real-time contributions via M-Pesa integration.

## Features
- **Campaign Tracking**: Visual progress bars and real-time statistics for fundraising goals.
- **M-Pesa Integration**: Simulated STK push workflow for mobile payments.
- **Responsive Design**: Built with Tailwind CSS for mobile and desktop compatibility.
- **TypeScript Safety**: Fully typed data structures for campaigns and contributions.

## Getting Started
1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Build for production: `npm run build`

## Tech Stack
- React 18
- TypeScript
- Tailwind CSS
- Lucide Icons
- Vite
